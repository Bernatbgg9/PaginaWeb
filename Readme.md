# Dropshipping Backend (FastAPI)

API backend para una tienda de dropshipping.

## Instalación

```bash
git clone https://github.com/TU_USUARIO/PaginaWeb.git
cd PaginaWeb
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Ejecutar servidor

```bash
uvicorn app.main:app --reload
```
